# Faculty Personal Dashboard

A prototype web application for faculty at Batangas State University-Lipa Campus to track personal food consumption and transportation activities. This is a faculty copy of the student prototype adjusted with faculty-specific storage keys.

## Features

- Food consumption tracking
- Transportation tracking with CO₂ calculations
- Daily and weekly summaries, charts, and CSV export
